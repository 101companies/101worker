The SoLaSoTe ontology

Status
Unpublished manuscript.

Authors
Ralf Lämmel and Martin Leinberger and Andrei Varanovich

Abstract
This document describes and exercises the SoLaSoTe ontology for software languages, software technologies, and software concepts, as they are relevant for programming and software engineering. Evidence from the software chrestomathy of the 101 project is used to demonstrate the ontology.
